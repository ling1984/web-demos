# Day-N-Night

Short animation of a day/night cycle made with HTML5 Canvas.

Features (Implemented)
* Sun/ moon path
* Sky colour change
* Colour change as a gradient

Features (Not yet)
* Stars - brightness change with night sky
* Spots on the moon
* Crescent moon
* Sunrise/ Sunset
* Clouds
* Ground
* Trees
* Hills
* Trees Growing
* Summer/ winter/ spring/ autumn
